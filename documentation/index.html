Como instalar o modelo

Installing a nuron template is not like WordPress or CMS theme installation, actually, installation keyword does not go with nuron template. Why? because you don’t install anything, yes nuron or HTML templates are automatically rendered by the browser.

1. Editando o modelo

Para isso, você precisará de um editor de código, como VScode, Sublime Text, etc. Quando a edição do modelo estiver concluída usando o editor, salve os arquivos e a pasta e prossiga com o upload dos arquivos do modelo no servidor ativo.

2. Converter LTR para RTL

Se você deseja converter seu modelo html padrão para RTL, então você deve fazer estas três alterações em todos os arquivos html.

Passo 1: Adicione o atributo RTL à tag HTML: 

                                                
                                                    <!DOCTYPE html>
                                                    <html lang="en" dir="rtl">

Passo 2: Inclua o arquivo CSS RTL:

Certifique-se de vincular um arquivo CSS RTL no <head> seção do seu HTML

                                                
                                            <link rel="stylesheet" href="assets/css/rtl-style.css">

Passo 3: Inclua o arquivo Bootstrap RTL: 

                                                
                                            <link rel="stylesheet" href="assets/css/bootstrap.rtl.min.css">

3. Para converter o modelo para o modo escuro

Passo 1: Inclua o arquivo JavaScript:

4. Carregamento para o servidor ativo usando FTP:

Primeiro, se você não tiver o modelo no seu computador, baixe o modelo para começar. Quando o download estiver concluído, você receberá um pacote como este da captura de tela (após descompactar).

5. O pacote vem com documentação. Descompacte o pacote baixado e você encontrará uma pasta com todos os arquivos de modelo, como na captura de tela acima .

Alterar imagens

Para alterar qualquer imagem do site

  1. Recolha cuidadosamente o nome da fonte da imagem (ou seja, banner-img.jpg)
  2. Abra o arquivo chamado imagens.
  3. Encontre o arquivo de imagem específico.
  4. Substitua o arquivo pela sua imagem.
  5. Certifique-se de que o nome do arquivo não seja alterado. O nome do arquivo deve ser o mesmo.

Exemplo: Suponha que você queira alterar o seguinte arquivo de imagem:

                                                    
                                                        <div class="logo">
                                                            <a href="index.html">
                                                                <img src="assets/images/logo/logo.png" alt="Education Logo Images">
                                                            </a>
                                                        </div>

Você deve fazer o seguinte para alterar esta imagem com sua própria imagem:

  1. Abra o arquivo chamado imagens.
  2. Encontre o arquivo de imagem específico chamado banner-img.jpg
  3. Substitua o arquivo pela sua imagem
  4. Certifique-se de que o nome do arquivo não seja alterado.
Observação

As imagens vistas na pré-visualização ao vivo são apenas para fins de pré-visualização. Elas não estão incluídas nos arquivos de download.
